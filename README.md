**HR Analytics Dashboard
Project Overview**

The HR Analytics Dashboard is an interactive data visualization project developed using Microsoft Power BI. The dashboard analyzes employee data to provide insights into workforce trends such as attrition, salary distribution, job satisfaction, and employee demographics. It helps HR teams and management make data-driven decisions for improving employee retention and workforce planning.

**Key Features**

1.Employee Attrition Analysis

2.Workforce Demographics Overview

3.Salary Distribution Insights

4.Job Satisfaction Analysis

5.Department-wise Employee Distribution

6.Interactive Filters for Gender and Department

**Key Metrics**

1.Total Employees

2.Attrition Count

3.Attrition Rate

4.Average Age

5.Average Salary

6.Job Satisfaction Score

**Tools & Technologie**s

1.Microsoft Power BI

2.Microsoft Excel

3.Data Visualization

4.Data Modeling

5.DAX Calculations

**Dataset Information**

The dataset contains employee information including:

1.Employee ID

2.Department

3.Job Role

4.Gender

5.Age

6.Salary

7.Education

8.Job Satisfaction

9.Attrition Status
